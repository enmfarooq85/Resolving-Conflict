# Git Merge Conflict (Simple Explanation)

---

## What is a Merge Conflict?

A merge conflict happens when Git cannot automatically combine changes from two branches.

This usually occurs when:
- Same file is changed in both branches
- Same lines are edited differently
- Git does not know which version to keep

---

## Why Merge Conflicts Happen?

Git tries to merge automatically, but fails when:

- Two developers edit the same part of code
- One branch deletes a line and another modifies it
- Same function/logic is changed differently
